# CineTicket - Movie Ticket Booking System

A premium, feature-rich Movie Ticket Booking System built with **ASP.NET Web Forms**, **ADO.NET**, and **SQL Server**. This system provides a seamless experience for both moviegoers and administrators, featuring a modern dark-themed UI and professional ticket printing capabilities.

## 🚀 Key Features

### 🎟️ For Users
- **Dynamic Movie Catalog**: Browse the latest releases with high-quality posters and detailed information.
- **Seat Selection & Booking**: Intuitive booking flow with real-time seat tracking.
- **My Bookings**: Manage your viewing history and active tickets.
- **Professional PDF Tickets**: Generate and print premium-designed tickets with unique QR codes for every booking.
- **User Profile Dashboard**: Personalized welcome screens and account management.

### 🛠️ For Administrators
- **Admin Dashboard**: Real-time stats on movies, users, and revenue.
- **Advanced User Management**: Full control over user accounts, roles, and password resets.
- **Movie & Showtime Management**: Add, edit, and schedule movies with ease.
- **Booking Logs**: Monitor all transactions and manage cancellations.
- **AJAX-Powered UI**: Smooth, non-page-reloading interactions for a better management experience.

## 💻 Tech Stack
- **Backend**: C# (ASP.NET Web Forms)
- **Database**: SQL Server (ADO.NET)
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript, Bootstrap 5
- **Icons**: FontAwesome 6
- **QR Code API**: QRServer API

## 🎨 Design Philosophy
- **Modern Aesthetics**: Sleek dark mode by default with vibrant accents.
- **Glassmorphism**: Elegant card designs with subtle blur and shadow effects.
- **Responsive Layout**: Fully optimized for various screen sizes.
- **Print Optimization**: Dedicated `@media print` styles for high-quality ticket generation.

## 🛠️ Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/MaingiJulius/Movie-Ticket-Booking-System.git
   ```
2. Open the solution in **Visual Studio**.
3. Configure your SQL Server connection string in `Web.config`.
4. Run the `seed_movies.sql` script to initialize the database with sample data.
5. Press `F5` to build and run the project.
